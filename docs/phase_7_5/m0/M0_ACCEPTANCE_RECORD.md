> Accepted direction: **Option C — Executive Warmth / الدفء الإداري**

## Accepted Product and Visual Contract

- Warm stone/cream light shell with calm rounded surfaces.
- Restrained Brand Gold and separate accessible Action Gold roles.
- Option B-inspired compact filter/table density for Finance and Inventory.
- Three-layer navigation: module sidebar, global top bar, contextual module
  navigation.
- Locked primary module order and lower Audit/Settings/user area.
- Four fixed Dashboard v1 priorities.
- Permission-safe global search, quick create, notifications, Dashboard,
  Daily Activity, Audit, and drill-down behavior.
- Noto Sans/Noto Sans Arabic remains the Phase 7.5 baseline.
- Dark mode remains deferred to Phase 12.
- Existing URLs remain valid while M2 introduces the accepted module shell.
